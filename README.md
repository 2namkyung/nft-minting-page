# NFT 민팅

## 민팅하기

- 지정된 블록 넘버가 지나면 민팅가, 민팅할 수를 정하여 트랜잭션을 보낼 수 있다
- 리액트 타입스크립트 연습
- ethers를 이용한 WebSocketProvider로 블록 넘버 가져오기
- 메타마스크 연결하기
- Recoil을 이용한 지갑 연결 상태 관리하기
- tailwind를 이용하여 간단하게 꾸미기

#
